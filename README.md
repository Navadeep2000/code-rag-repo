# code-rag-repo

**Goal:** Build a simple system that retrieves relevant code snippets from your repo and uses an LLM to answer coding questions.

**Tech stack (simple & lightweight):**

- Python
- langchain or llama-index (you pick)
- SentenceTransformers embeddings
- FAISS (simple, fast vector store)
- Open-source LLM via Ollama or OpenAI API (you choose)
